## Select text and graphics

> [!tip]
> **Tip:** If you know the key combination to move the cursor, you can generally select the text by using the same key combination while holding down Shift. For example, ⌘+▶︎ moves the cursor to the next word, and ⌘+Shift+▶︎ selects the text from the cursor to the beginning of the next word.

- Use `cmd` + `E` toggle between reading and editing view.

- Press `shift` + `enter` to create a line break within a block of text.
- Press `cmd` + `/` to create a comment.
- Type `---` or `***` to create a divider. (Three dashes in a row.)
- Press `cmd` + `shift` + `M` to create a comment.
- With text selected, press `cmd` + `B` to **bold text**.
- With text selected, press `cmd` + `I` to *italicize text*.
- With text selected, press `cmd` + `U` to underline text.
- With text selected, press `cmd` + `shift` + `S` for strikethrough.
- With text selected, press `cmd` + `K` to add a link. You can also paste a URL over selected text to turn it into a link using `cmd` + `V`.
- With text selected, press `cmd` + `E` for inline code.
- Press `tab` to indent and nest content.
- Press `shift` + `tab` to un-nest content.

For all of these content creation shortcuts, `cmd` + `option` is the combination for Mac, and `ctrl` + `shift` is the combination for Windows and Linux:
